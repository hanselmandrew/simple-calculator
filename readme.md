# Basic Calculator App

Takes expressions in plain text form and prints the result. My first stab at writing an interpretter and expression tree evaluator.

## Currently Supports:

1. Whitespace is ignored
2. Operators:
    1. \+
    2. \-
    3. \*
    4. \/
3. Order of operations:
    1. */
    2. +-
